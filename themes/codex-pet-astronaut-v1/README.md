# Codex Pet Astronaut

Transparent PNG assets for a desktop coding assistant pet.

## States

| State | Asset | Suggested usage |
| --- | --- | --- |
| `idle` | `idle.png` | Waiting for a task |
| `thinking` | `thinking.png` | Planning or reasoning |
| `coding` | `coding.png` | Editing files or running commands |
| `success` | `success.png` | Task completed successfully |
| `error` | `error.png` | An error occurred or user input is needed |

Use `manifest.json` to map application states to assets.
